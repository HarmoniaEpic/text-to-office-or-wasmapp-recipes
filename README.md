# 🎉 かんたん！Office文書の自動生成ガイド

## これは何？

**AIに頼むだけで、PowerPoint・Word・Excel・PDFファイルが自動で作れる**便利な機能です！

4つのレシピファイル（PPTX・DOCX・XLSX・PDF）を使って、AIがあなたの要望通りのOffice文書やPDFを生成します。

面倒な作業はAIにお任せして、あなたは内容を考えるだけでOK。

## 🚀 4ステップで完成！

### 1️⃣ 作りたいファイルに合わせてレシピを選ぶ
- **PowerPointを作る** → `PPTX-RECIPE.md`を選ぶ
- **Wordを作る** → `DOCX-RECIPE.md`を選ぶ
- **Excelを作る** → `XLSX-RECIPE.md`を選ぶ
- **PDFを作る** → `PDF-RECIPE.md`を選ぶ

### 2️⃣ 選んだレシピをAIに添付する
※ Excelで既存データを使う場合は、CSVファイルも一緒に添付

### 3️⃣ AIに作りたいものを伝える
### 4️⃣ 生成されたHTMLを保存して開き、ダウンロード

たったこれだけ！

## 📎 レシピファイルの使い方

### 🔍 **どのレシピを使えばいい？**

| 作りたいもの | 使うレシピファイル | ファイル名 |
|------------|-----------------|-----------|
| 📊 プレゼン・スライド | PowerPointレシピ | `PPTX-RECIPE.md` |
| 📄 文書・レポート | Wordレシピ | `DOCX-RECIPE.md` |
| 📈 表・グラフ・データ | Excelレシピ＋データCSV（任意） | `XLSX-RECIPE.md` + データ.csv |
| 📑 請求書・証明書・印刷物 | PDFレシピ | `PDF-RECIPE.md` |

### 📤 **AIへの添付方法**

1. **作りたいファイルの種類を決める**
   - 例：「売上報告のプレゼンを作りたい」→ PowerPoint

2. **対応するレシピファイルを選ぶ**
   - PowerPointなら → `PPTX-RECIPE.md`

3. **AIとの会話で添付**
   - ファイルをドラッグ＆ドロップまたは📎ボタンで添付
   - レシピファイルを添付してから、作りたい内容を伝える
   - **Excelの場合**: データがあればCSVファイルも一緒に添付

### 📝 **CSVファイルの例**
```csv
商品名,数量,単価,売上
ノートPC,5,80000,400000
マウス,20,3000,60000
キーボード,15,5000,75000
```
このようなCSVファイルを添付すると、AIがデータを読み込んでExcelファイルを作成します。

### ✅ **正しい依頼の流れ**

**PowerPoint/Wordの場合：**
```
【手順1】PPTX-RECIPE.mdを添付
【手順2】「営業報告のプレゼンを5枚で作ってください」と依頼
【手順3】AIがHTMLファイルを生成
【手順4】HTMLを保存して開く → ダウンロード！
```

**Excel（データ新規作成）の場合：**

```
【手順3】AIがHTMLファイルを生成
【手順4】HTMLを保存して開く → ダウンロード！
```

**Excel（既存データ使用）の場合：**
```
【手順1】XLSX-RECIPE.mdを添付
【手順2】売上データ.csvを添付
【手順3】「CSVデータから分析レポートを作ってください」と依頼
【手順4】AIがHTMLファイルを生成
【手順5】HTMLを保存して開く → ダウンロード！
```

**PDFの場合：**
```
【手順1】PDF-RECIPE.mdを添付
【手順2】「請求書のPDFを作ってください」と依頼
【手順3】AIがHTMLファイルを生成
【手順4】HTMLを保存して開く → ダウンロード！
```

## 💬 AIへの頼み方（コピペOK！）

### ⚠️ **重要：まずレシピファイルを添付！**
以下の依頼をする前に、必ず対応するレシピファイルを添付してください：
- PowerPoint → `PPTX-RECIPE.md`を添付
- Word → `DOCX-RECIPE.md`を添付
- Excel → `XLSX-RECIPE.md`を添付

### PowerPointを作りたいとき

**【手順】**
1. `PPTX-RECIPE.md`を添付
2. 以下のメッセージを送信

```
新商品の紹介プレゼンを作ってください。
以下の構成でお願いします：
1. タイトル「新商品のご案内」
2. 商品の特徴（3つ）
3. 価格とプラン
4. お問い合わせ先
```

### Wordを作りたいとき

**【手順】**
1. `DOCX-RECIPE.md`を添付
2. 以下のメッセージを送信

```
会議の報告書を作成してください。
日付：2025年1月17日
参加者：5名
議題：今年度の計画について
決定事項と次回までの課題を含めてください。
```

### Excelを作りたいとき

**【手順】**
1. `XLSX-RECIPE.md`を添付
2. データがある場合は、CSVファイルも一緒に添付（オプション）
3. 以下のメッセージを送信

#### 📊 **データを新規作成する場合**
```
1月の売上管理表を作ってください。
日付、商品名、数量、金額の列を作って、
合計が自動計算されるようにしてください。
グラフも付けてください。
```

#### 📈 **既存のデータを使う場合**
```
【CSVファイルも一緒に添付して】
添付したCSVデータを使って、見やすいExcelレポートを作ってください。
・データを表形式に整形
・合計と平均を自動計算
・グラフを追加
・条件付き書式で重要な値を強調
```

💡 **CSVファイルの準備方法**
- メモ帳やExcelで作成可能
- 1行目に項目名（例：日付,商品名,数量,金額）
- 2行目以降にデータを記入
- カンマ（,）で区切って保存

### PDFを作りたいとき

**【手順】**
1. `PDF-RECIPE.md`を添付
2. 以下のメッセージを送信

```
請求書のPDFを作成してください。
宛先：〇〇株式会社
品目：
- 商品A 10個 単価1,000円
- 商品B 5個 単価2,000円
消費税10%を含めた合計金額を計算し、
振込先情報も記載してください。
```

## ❓ よくある質問

### Q: レシピファイルはどこで入手できますか？
**A:** 4つのレシピファイル（PPTX-RECIPE.md、DOCX-RECIPE.md、XLSX-RECIPE.md、PDF-RECIPE.md）は本READMEとセットで提供されています。

### Q: レシピファイルを添付し忘れたらどうなりますか？
**A:** AIが正しくファイルを生成できません。必ず最初にレシピファイルを添付してから依頼してください。

### Q: ExcelでCSVファイルは必須ですか？
**A:** いいえ、CSVファイルは任意です。既存のデータを使いたい場合のみ添付してください。データを新規作成する場合は不要です。

### Q: CSVファイルの作り方が分かりません
**A:** メモ帳を開いて、1行目に項目名（例：商品名,価格,数量）、2行目以降にデータを入力し、「.csv」の拡張子で保存すればOKです。Excelで作成して「CSVとして保存」でも作れます。

### Q: 無料で使えますか？
**A:** はい！生成されたファイルは無料でダウンロードできます。

### Q: スマホでも使えますか？
**A:** パソコンでの利用を推奨します。スマホでは一部機能が制限される場合があります。

### Q: 作成に時間はかかりますか？
**A:** 通常10秒程度で完成します。

### Q: 後から編集できますか？
**A:** はい！ダウンロードしたファイルは、Office製品で自由に編集できます。

### Q: 文字化けしたら？
**A:** 最新のOffice製品またはGoogleドキュメント等で開いてください。
